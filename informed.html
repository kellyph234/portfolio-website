inFormed

What Inspired inFormed?

inFormed was created from a CareerFoundry assignment requiring us to design an iOS and Android application. With the current political climate, I wanted to find something that allowed me to follow causes I cared about, and showed me ways I can become active

My Process

Key Features

  • Users can follow issues state by state
  • A place where users can sign petitions for causes they follow
  • A feature that allows people to see opposing sides within an article
  • Users can contact their reps via the app about current issues

The Initial Design

I created the initial wireframes for both iOS and Android screens. Using these simple frames, I was able to create mid-fidelity frames that incorporated iOS and Android features

Mid-Fidelity Frames

I added basic features to distinguish between iOS screens and the Material Design screens. Some of these things included the iOS home bar, iOS navigation bar, Android navigation bar and Android status bar

User Testing

Before creating the final screens, I crowd sourced a pool of candidates that were both iOS and Android users to provide critical feedback for each screen. From this feedback, I was able to add better features to distinguish between each screen
